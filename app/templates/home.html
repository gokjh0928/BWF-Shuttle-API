BWF Shuttle API

The most comprehensive and detailed source to view and download top badminton player rankings and data specified by the Badminton World Federation.

About

BWF Shuttle API is a Flask Web Application that utilizes Python libraries BeautifulSoup and Pandas to scrape badminton ranking da. CSV and JSON files can be downloaded directly, and JSON metadata can be acquired through API calls.

API Call Formats

Fill in the { } from the choices below, for the data you wish to get. For months and days, include the 0 at the start if it's a single digit.

http://bwfshuttleapi.herokuapp.com/rankings/api/{category}

http://bwfshuttleapi.herokuapp.com/rankings/api/{category}/{rows}

http://bwfshuttleapi.herokuapp.com/rankings/api/{category}/{year}/{week}/{rows}

http://bwfshuttleapi.herokuapp.com/rankings/api/{category}/{year}/{month}/{day}/{rows}

Purpose

As of this moment, no up-to-date official database for badminton players is currently available to the public to download from. BWF Shuttle API acts as an alternative source for anyone wanting to view or download BWF ranking data.
