BWF Shuttle API

The most comprehensive and detailed source to view or download top badminton player rankings and data specified by the Badminton World Federation.

About

BWF Shuttle API is a Flask Web Application that utilizing Python libraries BeautifulSoup and Pandas to scrape data from the official BWF website. CSV and JSON files can be downloaded directly, and JSON metadata can be acquired through API calls.

API Call Formats

Categories: MS, WS, MD, WD, XD

http://bwfshuttleapi.herokuapp.com/rankings/api/{category}

http://bwfshuttleapi.herokuapp.com/rankings/api/{category}/{rows}

http://bwfshuttleapi.herokuapp.com/rankings/api/{category}/{year}/{week}/{rows}

http://bwfshuttleapi.herokuapp.com/rankings/api/{category}/{year}/{month}/{day}/{rows}

Purpose

As of this moment, no official database for badminton players is currently available to the public to download from. BWF Shuttle API is an alternative source for anyone wanting to view or download BWF ranking data.
